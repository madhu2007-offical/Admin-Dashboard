# Experiment 1.3: Admin Dashboard

## 1. Aim
To develop an admin dashboard with CSS Grid layout and dynamic theme switching.

## 2. Objective
- Create grid-based responsive layout   
- Implement dark/light mode toggle  
- Use CSS variables for theme management  
- Persist theme preference using `localStorage`  

## 4. About the Program
This experiment covers CSS Grid template areas, CSS variables, and JavaScript class manipulation for theme switching.  
It helps in designing responsive admin dashboards and managing dynamic UI themes efficiently.

**Output:**  
A functional admin dashboard with a responsive grid layout and dark/light mode toggle feature.

## 5. Learning Outcomes
After completing this experiment, students will be able to:  
- Understand and implement **CSS Grid layouts** for complex UI designs  
- Apply **CSS variables** for theming and maintainable styling  
- Use **JavaScript DOM manipulation** to switch themes dynamically  
- Persist user preferences using **localStorage**  
- Develop responsive and interactive **web dashboards**
